# Security policy

Thor is pre-1.0. Security fixes target the latest release only.

Please report suspected vulnerabilities privately through GitHub Security
Advisories for `gilvandovieira/thor`. Do not include credentials, production
data, or a public proof of concept in an issue. Include the affected version,
runtime/database, impact, and the smallest safe reproduction you can provide.

Public issues are appropriate for hardening ideas that do not disclose an
exploitable vulnerability.
